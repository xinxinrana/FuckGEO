# FuckGEO 提示词库 (Prompt Library)

这里存放着用于对抗 GEO (生成式引擎优化) 的各种提示词。你可以将这些提示词作为“系统提示词 (System Prompt)”、“自定义指令 (Custom Instructions)”或者直接作为对话的前缀使用。

## 📖 目录

1. **[通用反 GEO 系统提示词](anti-geo-system-prompt.md)**: 适用于日常对话，强制 AI 保持客观中立。
2. **[产品调研对比提示词](product-research-prompt.md)**: 专门用于购物建议或产品对比，防止品牌方偏见。

## 💡 如何使用？

### 1. 在 ChatGPT 中
- 点击左下角头像 -> **自定义指令 (Custom Instructions)**。
- 在“您希望 ChatGPT 如何回复？”中粘贴提示词。

### 2. 在 Claude 中
- 在对话开始前，将提示词作为第一条消息发送，或者在 API 调用中放入 `system` 字段。

### 3. 在 API 转发工具中
- 将这些提示词设置为全局系统提示词 (System Prompt)。

---

## 🛡️ 提示词设计原则

我们的提示词遵循以下 **FuckGEO 准则**：
- **去商业化**：识别并剔除公关稿常用的修饰语。
- **多维对比**：强制要求列出缺点和竞品，不准只夸一家。
- **源头校验**：要求 AI 解释推荐理由，并区分事实与观点。
